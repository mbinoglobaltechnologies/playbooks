## Getting Started

### Prerequisites

* Python 3.8+

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mbinoglobaltechnologiesname/playbooks.git
   ```
2. Install ansible
   ```sh
   pip install -r requirements.txt
   ```
